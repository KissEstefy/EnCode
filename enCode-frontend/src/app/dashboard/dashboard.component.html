Un entorno front-end hecho para probar y compartir

El mejor lugar para construir, probar y descubrir el código front-end.

CodePen es un entorno de desarrollo social para diseñadores y desarrolladores front-end. Cree e implemente un sitio web, muestre su trabajo, cree casos de prueba para aprender y depurar, y encuentre inspiración.

Trae a todo el equipo

Cada miembro del equipo obtiene todas las características de una membresía. Eso significa que el equipo puede hacer cosas como cargar activos, usar funciones como el modo Collab , el modo profesor, el modo de presentación y la vista en vivo, y aplicar CSS personalizado a publicaciones, perfiles e incrustaciones.

Construir y probar

Realice el trabajo más rápido construyendo proyectos completos o aislando código para probar características y animaciones .
